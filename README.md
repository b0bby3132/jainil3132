# Riipen Capstone Project: Employee Engagement Dashboard

##  About the Project
This project was completed as part of a Riipen capstone collaboration with Carmela Industries. The goal was to design and develop an automated dashboard system to analyze employee engagement and support data-driven decision-making.

The dashboard focuses on key areas such as **Happiness, Personal Motivation, Company Motivation, and Workplace Relationships**. Synthetic data was used to simulate real survey responses and build a scalable solution.

---

##  My Role
I was responsible for **automating the data flow from survey responses to the dashboard**. My contributions included:

- Connecting Google Forms data to Excel and Power BI  
- Cleaning and preparing data for analysis  
- Transforming data using Power Query  
- Supporting the development of role-based dashboards  
- Helping design a system that updates automatically  

---

##  Key Skills & Tools
- **Power BI** → Dashboard creation and visualization

##  Project Highlights
- Built an automated workflow: **Google Forms → Excel → Power BI**  
- Created dashboards for different users (leaders, employees, departments)  
- Identified trends in employee engagement over time  
- Highlighted differences in engagement across departments  
- Designed a flexible system that works even when survey questions change
